# 브라우저 게임 모음

브라우저에서 바로 실행할 수 있는 작은 게임 모음입니다.

## 실행

로컬 서버를 띄운 뒤 아래 주소로 열면 됩니다.

- 수박게임: `http://127.0.0.1:4173/`
- 2048: `http://127.0.0.1:4173/2048.html`

## 조작

### 수박게임

- 마우스/터치: 위치를 잡고 클릭 또는 터치해서 과일 떨어뜨리기
- 키보드: `←`, `→` 이동, `Space` 또는 `Enter` 떨어뜨리기
- `P`: 일시정지
- `R`: 다시 시작

### 2048

- 키보드: `←`, `→`, `↑`, `↓` 또는 `WASD`
- 터치: 보드에서 스와이프
- 화면 버튼: `←`, `→`, `↑`, `↓`
- `R`: 새 게임
- `U`: 되돌리기
