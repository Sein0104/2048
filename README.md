# 2048

브라우저에서 바로 실행할 수 있는 2048 게임입니다.

## 실행

로컬 서버를 띄운 뒤 아래 주소로 열면 됩니다.

- 2048: `http://127.0.0.1:4173/`

## 조작

- 키보드: `←`, `→`, `↑`, `↓` 또는 `WASD`
- 터치: 보드에서 스와이프
- 화면 버튼: `←`, `→`, `↑`, `↓`
- `R`: 새 게임

## 랭킹

게임오버 후 닉네임을 입력하면 Supabase의 `scores` 테이블에 점수가 저장됩니다. 배포 전에 `supabase-scores.sql`을 Supabase SQL Editor에서 실행해야 공용 순위가 동작합니다.
